# Simple WallpaperApp
This is a simple demo wallpaper app which contains wallpaper links (json file) in assets folder.
- Download Wallpaper ⬇️
- Sort 🔎 by Collections
- Clean UI 💚
- Light/Dark 🌗 Theme
